Installing and using s3cmd

Download Go to http://sourceforge.net/projects/s3tools/files/s3cmd/ , select the latest version folder and download the tar.gz file. Unzip it with tar -zxvf <filename> Install First make sure you have python setup tools installed. To install run: sudo apt-get install python-setuptools Then go inside your unpacked previously directory and run: sudo python setup.py install And that’s it for installing. Now configuring and using: Configuring To configure, run: s3cmd --configure You will be prompted for Access key and Secret Key.
