Useful Git commands

Initialize git in a working directory So you have a directory where you want to set up git in. You cd into that directory and you do git init That’s all it takes to have git initialized in that directory. Connect to a remote repository Now what you want is to connect that git instance to a remote branch. To do this run: git remote add origin <URL> Now you’re connected to a remote repo.
